# CODSOFT — Artificial Intelligence Internship

This repository contains my submissions for the **CodSoft AI Internship**.
As per the internship guidelines, at least 3 tasks must be completed. I have
completed the following:

| Task | Title | Folder |
|------|-------|--------|
| 1 | Rule-Based Chatbot | [`Task1_Chatbot`](./Task1_Chatbot) |
| 2 | Tic-Tac-Toe AI (Minimax) | [`Task2_TicTacToe_AI`](./Task2_TicTacToe_AI) |
| 4 | Recommendation System | [`Task4_Recommendation_System`](./Task4_Recommendation_System) |

## Dashboard

[`dashboard.html`](./dashboard.html) is a single-file, self-contained dashboard
with a live, playable demo of all three tasks (chatbot, tic-tac-toe AI, and
recommender) reimplemented in JavaScript. Open it directly in any browser —
no server or build step required. Great for the demo video / LinkedIn post.

## About

CodSoft is a community-driven organization that offers internships in various
domains including Web Development, Android Development, Python Programming,
Java Programming, and Artificial Intelligence. This repo documents my work
for the AI track.

## How to run

Each task folder has its own `README.md` with setup and run instructions.
In general:

```bash
git clone https://github.com/agpriyanshi1111-debug/CODSOFT.git
cd CODSOFT/<task-folder>
pip install -r requirements.txt   # if present
python <script_name>.py
```
